TypeScript Gatsby and Tailwind CSS (using emotion) starter kit. [Tailwind CSS](https://tailwindcss.com/) is a CSS framework that provides
basic utilities for building custom designs.
This [Gatsby.js](https://www.gatsbyjs.com/) boilerplate comes with Tailwind CSS support by using
[`twin.macro`](https://github.com/ben-rogerson/twin.macro) and [emotion](https://github.com/emotion-js/emotion) to handle underlying css-in-js.

- Typical Gatsby bootstraped features like [Prettier](https://prettier.io/) & [ESLint](https://eslint.org/)
- TypeScript
- Aliased imports `~/components` -> `./src/components`
- Automatically generated TypeScript types for your GraphQL queries available at `./gatsby-graphql.ts`.
- Build components utilizing [Tailwind CSS](https://tailwindcss.com/) class names via [`twin.macro`](https://github.com/ben-rogerson/twin.macro)
and powered by [emotion CSS-in-JS](https://github.com/emotion-js/emotion).
  You can use `css` prop

  ```jsx
  import React from "react";
  import tw from "twin.macro"; // imported function

  const SomeComponent: React.FC = ({children }) => {
    // we are using emotion css pragma with `tw` here
    return <div css={tw`container py-3`}>{children}</div>
  }
  ```

  ***or***

  You can use the custom provided `tw` pragma prop but you will need to add an import
  ```jsx
  import React from "react";
  import "twin.macro";

  const SomeComponent: React.FC = ({children }) => {
    // we are using special pragma `tw` here
    return <div tw="container py-3">{children}</div>
  }
  ```
